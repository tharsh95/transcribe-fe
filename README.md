## Getting Started

Follow the steps below to clone the repository, install dependencies, and start the development server:

1. **Clone the Repo**
   ```bash
   git clone <repository-url>
   cd <repository-folder>
    ```
2. **Install dependenncies**
   ```bash
   npm install
    ```
3. **Start the server**
   ```bash
   npm run dev
   ```
4. **Visit in Browser**

Open your browser and go to:
http://localhost:5173
